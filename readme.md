Overview

PHP + MySQL web application for managing books

Twig template engine for page rendering

AJAX autocomplete for fast searching

Fully responsive layout

Hosted on mi-linux server

Live site: https://mi-linux.wlv.ac.uk/~2420693/book/public/

Features

Full CRUD operations (add, view, edit, delete)

User registration and login

Password hashing for security

Session-based page protection

Math CAPTCHA for login and registration

Multi-criteria search (title, genre, year)

AJAX autocomplete search

Server-side validation & output escaping

Installation

Import users and books tables into MySQL

Update config.php with correct credentials

Run locally at:

http://localhost/bookstore/public/


Upload project into public_html on mi-linux

Usage

Register or log in

Add, edit, delete, or search books

Autocomplete suggests titles dynamically
